Quick reference guide for HTML and CSS




Part one: HTML



Introduction to HTML

  • HTML (Hypertext Markup Language) is the standard markup language used to create web pages.
  • HTML is the backbone of the web, and with this quick reference guide, creators could understand some ofthe most essential HTML concepts, including basic structure, headings, paragraphs, links, lists, and tables, etc.

In this quick reference guide, I will be giving short description along with examples of these following html tags given in the table below:

Basic HTML tags and it's usage
1.<html>: The root element of the html document.
2.<head>: Contains meta information about the html page.
3.<title>: Name of a page displayed on the tab.
4.<body>: Contents.
5.<h>: Heading.
6.<p>: Paragraph.
7.<pre>: To display text with all empty spaces.
8.<break>: To insert a break line.
9.<o>: Hyperlink.
10.<img src>: To display image.
11.<li>: All items inside the list.
12.<ol>: Ordered list.
13.<ul>: Unordered list.
14.<table>: To display a table.

Basic Structures

HTML documents are made up of several elements, including the html, and body elements. In html, these elements are all written in [<>] brackets and they are called tags.

Tags act as controllers for all other html elements (except for the alt: |DOCTYPE> tag).

There are two tags for each element

  • opening tag
  • closing tag

For closing tag you need to add a '/'' in the start to indicate it is a closing tag.

Here's an example of a basic html document structure:

HTML
<!DOCTYPE html>
<html>
<head>
<title> My website name </title>
</head>
<body>
insert content here
</body>
</html>

The basic structure of an HTML document includes <!DOCTYPE html> declaration, <html> element, <head> element, and <body> element.

  1. The <!DOCTYPE html> tag is the declaration which defines that this document is an HTML document and it has no closing tag.
  2. The <html> tag is the root element of an HTML page.
  3. The <head> tag contains meta information about the HTML page.
